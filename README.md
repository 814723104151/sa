# 🧋 Stea – Bubble Tea Shop Website

## 📌 Project Overview
Stea is a modern and aesthetic Bubble Tea Shop website designed using HTML, CSS, and JavaScript.  
The website showcases different bubble tea flavors with images, prices, and descriptions in a clean card-style layout.

This project focuses on:
- Responsive Web Design
- Navigation Bar Implementation
- Image Card Layout Styling
- JavaScript Interaction (Sliding Menu)
- Smooth Scrolling Navigation

---

## 🌐 Features

✔ Navigation Bar (Home, About Us, Menu, Contact, Location)  
✔ Smooth Scroll Effect  
✔ About Us Section with Shop Description  
✔ Menu Section with 5 Tea Cards  
✔ Image + Description + Price for Each Tea  
✔ JavaScript Toggle Menu Function  
✔ Hover Animation Effects  
✔ Responsive Layout  
✔ Google Maps Location Link
